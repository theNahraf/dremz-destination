# 🌍 Dremz Destination - Travel Agency Website

Welcome to the frontend repository for **Dremz Destination**. This is a fully responsive, multi-page website built to generate leads, showcase premium tour packages, and establish brand trust.

## 🚀 Technology Stack
* **HTML5 & CSS3**
* **Tailwind CSS (via CDN)** for rapid, utility-first styling.
* **Vanilla JavaScript (ES6)** for dynamic content rendering and logic.
* **Swiper.js** for touch-friendly image carousels.
* **AOS.js** for smooth, scroll-triggered animations.
* **Font Awesome** for scalable vector icons.
